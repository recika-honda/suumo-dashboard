# Facility Logos

Brand logos used for shuhen (surrounding environment) photos.
When a known brand is detected, its logo is used instead of a Google Maps photo.

## File format
- JPEG or PNG, any size (auto-resized to 1280x960 white background)
- Filename must match the key in BRAND_LOGOS mapping in `skills/google-images.js`

## Adding a new logo
1. Save the logo file here (e.g., `seven-eleven.png`)
2. Add an entry in BRAND_LOGOS in `skills/google-images.js`

## Current mappings (logos to be added manually)
See BRAND_LOGOS in `skills/google-images.js` for the full list.
